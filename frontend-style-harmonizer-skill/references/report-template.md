## 报告字段说明

| 字段 | 来源 | 说明 |
|------|------|------|
| `{{date}}` | 当前时间 | 报告生成时间 |
| `{{path}}` | 用户输入 | 项目路径 |
| `{{scope}}` | 用户输入 | 审查维度 |
| `风险` | 自动判定 | 低/中/高，基于改动影响范围 |
| `出现次数` | 扫描统计 | 该裸值在项目中出现次数 |
| `收敛目标` | 算法推荐 | 建议的统一值 |
