# 个人中心页面模板

> uniapp个人中心页面，4套风格可选

## 目录结构

```
profile/
├── style-1-simple.vue    # 风格1：简约现代
├── style-2-vibrant.vue  # 风格2：活力炫彩
├── style-3-premium.vue  # 风格3：高端商务
└── style-4-nature.vue   # 风格4：清新自然
```

## 风格预览

### 风格1：简约现代
- 渐变头部背景
- 白色卡片
- 简洁布局
- 适合工具类App

### 风格2：活力炫彩
- 橙色渐变头部
- 圆角卡片
- 活跃氛围
- 适合电商/社交

### 风格3：高端商务
- 深色背景 (#0A0A12)
- 金色点缀 (#D4AF37)
- 会员尊贵感
- 适合金融/高端

### 风格4：清新自然
- 浅绿背景 (#F5FBF8)
- 绿卡会员卡片
- 柔和风格
- 适合健康/母婴
