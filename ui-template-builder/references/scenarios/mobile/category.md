# 分类页面模板

## 页面概述

商品分类导航页面，帮助用户快速找到目标商品。

## 适用场景

- 电商分类页
- 知识分类页
- 服务分类页

## 模板结构

```
category/
├── search-bar.vue      # 搜索栏
├── category-left.vue   # 左侧分类
├── category-right.vue  # 右侧商品
├── goods-list.vue     # 商品列表
└── mock.json
```

## 区域详解

### 1. 搜索栏

**位置**：页面顶部

**内容**：
- 搜索输入框
- 搜索图标
- 热门关键词（可选）

### 2. 左侧分类

**位置**：左侧固定区域

**宽度**：180-220px

**内容**：
- 分类名称
- 选中态高亮
- 可选图标

**交互**：
- 点击切换分类
- 右侧同步滚动

### 3. 右侧商品

**位置**：右侧区域

**内容**：
- 二级分类标题（可选）
- 商品网格/列表
- 分页加载

## 交互规范

1. **左侧滚动**：分类过多时独立滚动
2. **右侧联动**：切换分类时右侧滚动到顶部
3. **右侧滚动**：独立滚动，不影响左侧
4. **点击跳转**：点击商品跳转详情页
5. **下拉刷新**：刷新当前分类商品
