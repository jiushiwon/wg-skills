# 文章详情页模板

## 页面概述

展示文章详情内容的页面，支持目录导航、评论互动。

## 适用场景

- 新闻详情
- 博客文章
- 帮助文档
- 产品介绍

## 模板结构

```
article-detail/
├── header.vue         # 顶部信息
├── title.vue         # 标题区域
├── toc.vue           # 目录导航
├── content.vue       # 文章内容
├── author.vue        # 作者信息
├── comment.vue       # 评论区域
├── sidebar.vue       # 侧边信息
└── mock.json
```

## 区域详解

### 1. 顶部信息

**内容**：
- 分类/标签
- 发布时间
- 阅读量
- 来源

### 2. 标题区域

**内容**：
- 大标题
- 副标题
- 作者信息
- 发布时间
- 阅读/点赞/收藏

### 3. 目录导航

**位置**：左侧或右侧

**内容**：
- 自动提取标题
- 点击跳转
- 高亮当前阅读位置

### 4. 正文内容

**内容**：
- 段落
- 图片
- 代码块
- 引用
- 列表
- 表格

### 5. 底部信息

**内容**：
- 标签
- 分享
- 版权声明
- 相关文章
- 上一篇/下一篇

### 6. 评论区

**内容**：
- 评论列表
- 评论表单
- 点赞/回复
