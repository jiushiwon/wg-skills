# 注册页模板

## 页面概述

用户注册页面，支持多种注册方式和验证。

## 适用场景

- 用户注册
- 会员加入
- 企业入驻

## 模板结构

```
register/
├── register-form.vue   # 注册表单
├── agreement.vue       # 用户协议
├── third-party.vue    # 第三方注册
└── mock.json
```

## 注册方式

| 方式 | 字段 |
|------|------|
| 手机号注册 | 手机号 + 验证码 + 密码 |
| 邮箱注册 | 邮箱 + 验证码 + 密码 |
| 账号注册 | 用户名 + 密码 + 确认密码 |

## 验证规则

- 手机号格式
- 密码强度（6-20位）
- 两次密码一致
- 同意用户协议

## 风格

同登录页，保持一致。
