# UI Template Builder

> 页面模板固化技能：把常见页面固化下来，每次生成基于模板，确保效果稳定。

## 功能

- **页面模板**：覆盖移动端和PC端常见页面
- **风格选择**：每种页面提供 3-5 套风格
- **Mock数据**：一键生成配套Mock数据
- **技术栈适配**：uniapp Vue2/Vue3、React、Taro等

## 快速开始

```
# 生成个人中心页面（简约现代风格，带Mock数据）
生成一个个人中心页面

# 生成商品详情页（活力炫彩风格）
生成商品详情页，活力炫彩风格

# 生成PC登录页（深色科技风格，不带Mock）
做个登录页，暗黑科技风格，不需要mock数据
```

## 三个必须确认的配置

| 配置 | 默认值 | 说明 |
|------|--------|------|
| 技术栈 | uniapp Vue3 | 支持多种前端框架 |
| 页面类型 | 用户输入 | 从模板库选择 |
| 风格 | 简约现代 | 5套风格可选 |
| Mock数据 | 是 | 可选开启/关闭 |

## 组件模板库

组件模板与页面模板**相互独立**，页面模板可以直接引用或复制组件模板。

**不是 Element Plus / Ant Design / uView**：这里的组件是**纯模板代码**，无 npm 依赖、无额外编译配置、即拷即用。

### 移动端组件（uniapp Vue3）

| 组件 | 说明 |
|------|------|
| Button | 主/次/幽灵/危险按钮，多尺寸，加载态 |
| Modal | 弹框 / Dialog，确认/提示 |
| Drawer | 抽屉 / Sidebar，多方向滑出 |
| Toast | 轻提示，成功/错误/加载 |
| Picker | 底部选择器 |
| Calendar | 日历 / 日期选择 |
| Moments | 朋友圈 / 动态列表 |
| **Table** | 表格：列配置、排序、筛选、分页、斑马纹 |
| **Form** | 表单容器、表单项、输入框、选择器、校验规则 |

### PC端组件（React）

|------|------|

## 风格系统

| 风格 | 特点 | 适用场景 |
|------|------|----------|
| 简约现代 | 白色为主，文字为主 | 工具类App |

| 清新自然 | 绿/蓝色系，圆角卡片 | 健康/母婴 |
| 暗黑科技 | 深色背景，霓虹点缀 | 游戏/科技 |

## 与其他技能协作

| 技能 | 协作方式 |
|------|----------|

| impeccable | 生成后可通过impeccable优化 |

## 示例

### 示例1：生成个人中心

```
用户：做个个人中心页面

AI：
1. 技术栈确认（默认uniapp Vue3）
2. 页面类型：个人中心
3. 风格选择：
   - 简约现代（默认）
   - 活力炫彩
   - 高端商务
   - 清新自然
4. Mock数据：默认开启
5. 生成代码
```

### 示例2：生成数据大屏

```
用户：帮我做一个数据大屏，科技感强一点

AI：
1. 确认PC端数据大屏
2. 推荐风格：
   - 科技深蓝（推荐）
   - 赛博朋克
3. 开启Mock数据
4. 生成包含指标卡、折线图、饼图的完整大屏
```

---

**一句话**：选页面 → 选风格 → 确认Mock → 一键生成 ✓
