# UI Replica Skill 样例

本目录提供 3 个典型复刻场景的输出样例，用于展示 `ui-replica-skill` 的交付形态和还原度目标。

| 样例 | 场景 | 核心看点 |
|---|---|---|
| [admin-dashboard](admin-dashboard/) | 中后台管理系统 | Sidebar + Header + 卡片网格 + 步骤条 |
| [mobile-page](mobile-page/) | 移动端页面 | 底部 TabBar + 顶部导航 + 卡片列表 |
| [landing-page](landing-page/) | 营销落地页 | Hero + Features + CTA + Footer |

每个样例包含：

- `description.md`：原图结构分析
- `sample.html`：可直接浏览器打开的单文件 HTML

## 使用方式

1. 用浏览器直接打开 `sample.html` 查看效果
2. 对照 `description.md` 理解从图片到代码的拆解思路
3. 在真实复刻任务中参考这些样例的输出格式
