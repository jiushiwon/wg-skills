# 结构拆解模板

本模板用于 `ui-replica-skill` 把 UI 图片拆解为可编码的层级结构。

## 模板格式

```
页面: [页面名称]
类型: [中后台/营销/官网/移动端/文档/其他]
技术栈: [HTML+Tailwind/Vue3/React/uniapp/其他]

├── [区块 1: 名称]
│   ├── 尺寸: [宽 x 高 / 占满/固定]
│   ├── 背景: [颜色]
│   ├── 内边距: [数值]
│   └── 子元素:
│       ├── [元素 1]
│       └── [元素 2]
│
├── [区块 2: 名称]
│   ├── ...
```

## 示例

```
页面: 学术会议管理 - 二维码与接入
类型: 中后台管理系统
技术栈: HTML + Tailwind CSS

├── Sidebar (左侧边栏)
│   ├── 尺寸: 固定 232px 宽，占满高度
│   ├── 背景: #FFFFFF
│   ├── 边框: 右侧 1px #E5E7EB
│   └── 子元素:
│       ├── Logo 区 (高 64px)
│       ├── 导航菜单 (flex-1)
│       │   ├── 工作台
│       │   ├── 学术会议管理 (高亮态)
│       │   ├── 参会人员管理
│       │   └── ...
│       └── 底部操作区
│           ├── 发起会议按钮
│           ├── 使用手册
│           └── 退出登录
│
├── Header (顶部导航)
│   ├── 尺寸: 占满宽度，高 64px
│   ├── 背景: #FFFFFF
│   ├── 边框: 底部 1px #E5E7EB
│   └── 子元素:
│       ├── 左侧 tab 导航
│       └── 右侧工具区（通知、设置、用户信息）
│
├── Steps (步骤条)
│   ├── 背景: #FFFFFF
│   ├── 圆角: 12px
│   ├── 阴影: card
│   └── 子元素: 4 个步骤，前 3 完成，第 4 当前
│
└── Main Content (主内容网格)
    ├── 背景: #F3F4F6
    ├── 布局: 两栏，左 5/12，右 7/12
    └── 子元素:
        ├── 左侧：会议二维码卡片
        │   ├── 标题 + 自定义外观按钮
        │   ├── 二维码展示区（深色背景）
        │   ├── 说明文字
        │   └── 下载按钮组（PNG/JPG/PDF）
        │
        └── 右侧：3 个卡片
            ├── 外部平台集成
            ├── 准入与考勤控制
            └── 社交分享与链接
```

## 拆解原则

1. **先大块再小块**：先定 sidebar/header/main，再拆卡片和元素
2. **每个块只记关键信息**：尺寸、颜色、间距、位置
3. **不追求一次性精确**：先搭骨架，细节在编码阶段微调
4. **标注复用点**：多次出现的元素标记为公共组件
5. **标注特殊元素**：二维码、图表、地图、复杂动画等需要额外处理的元素

## 常见页面结构

### 中后台管理系统

```
页面
├── Sidebar
├── Header
└── Main
    ├── PageHeader（标题 + 操作按钮）
    ├── FilterBar（筛选/搜索）
    ├── Content（表格/卡片/表单）
    └── Pagination
```

### 营销落地页

```
页面
├── Navbar
├── Hero
├── Features
├── Testimonials
├── CTA
└── Footer
```

### 移动端页面

```
页面
├── TopNav
├── Content（可滚动）
└── TabBar
```
