# UI 复刻自检清单

执行 `ui-replica-skill` 时，按当前模式逐项检查。

## 快速模式检查项

### 读图

- [ ] 图片已读取成功，失败时要求用户重新提供
- [ ] 已判断页面类型（中后台/营销/官网/移动端/文档）
- [ ] 已画出页面结构树（至少两层）
- [ ] 已标记主色、文字色、背景色
- [ ] 已识别图标/图片/二维码/图表/地图等特殊元素

### 代码

- [ ] 使用单文件 HTML + Tailwind CDN 输出
- [ ] 先搭骨架再填血肉
- [ ] 颜色/间距/圆角已收敛到 Token
- [ ] 复杂元素优先用代码模拟，无法模拟再占位
- [ ] 输出完整可运行代码，可直接浏览器打开
- [ ] 已补全 hover/active/focus 等关键状态

### 交付

- [ ] 一句话结构分析
- [ ] 完整 HTML 代码
- [ ] 验证提醒：请用户打开查看并指出差异

## 完整模式检查项

### 项目与技术栈

- [ ] 已判断当前目录是否已有项目（仅作为提示）
- [ ] 已询问用户确认输出形态（HTML/Vue/React/uniapp/接入项目）
- [ ] 多图时已梳理页面关系

### 图质量评估

- [ ] 图片清晰度足够辨认文字和边界
- [ ] 图片包含完整页面或完整功能区块
- [ ] 关键元素无遮挡/水印
- [ ] 已判断是单屏截图还是长页面
- [ ] 长页面已按视觉断点拆分处理
- [ ] 质量不足时，已告知用户并请求补充

### 读图阶段

- [ ] 已识别页面类型
- [ ] 已画出页面结构树（至少两层）
- [ ] 已标记主色、文字色、背景色
- [ ] 已记录关键间距和尺寸
- [ ] 已识别图标来源或替代方案
- [ ] 已识别 banner/背景图/人物头像等视觉主体元素
- [ ] 已识别二维码/图表/地图等特殊元素并确定处理方案

### Token 阶段

- [ ] 颜色已收敛到主色 + 中性色阶梯
- [ ] 间距已收敛到 4 的倍数阶梯
- [ ] 圆角已收敛到 2-3 个等级
- [ ] 阴影已收敛到 1-2 个等级
- [ ] 字号已收敛到标题/正文/辅助三档以上

### 组件阶段

- [ ] 已识别可复用公共组件（仅用于当前页面组织）
- [ ] 已确定组件 props 接口
- [ ] 已区分公共组件和一次性样式

### 代码阶段

- [ ] 先搭骨架再填血肉
- [ ] 使用 Token 而非裸值
- [ ] 输出完整可运行代码
- [ ] 代码注释使用中文
- [ ] 不使用 `// ... rest of code`
- [ ] 已加入基础响应式断点

### 验证阶段

- [ ] 已补全 hover/active/focus/disabled 状态
- [ ] 已处理基础响应式
- [ ] 已按还原度目标自评
- [ ] 已说明未在浏览器实测
- [ ] 已建议用户打开查看并反馈差异

### 交付物

- [ ] 结构分析（一句话 + 结构树）
- [ ] 图质量评估结果
- [ ] Token 表
- [ ] 完整代码文件
- [ ] 验证提醒与还原度自评

## 增量精调检查项

- [ ] 只修改被指出的元素
- [ ] 已记录已确认的视觉参数
- [ ] 不重复输出整份 Token 表
- [ ] 输出修改后的完整文件或局部 diff
