# Agent 面试模拟题

一句话：高频 Agent 面试题及参考答案。

## 基础题

### Q1: 什么是 Agent？和传统 LLM 调用有什么区别？

**参考答案**：
Agent = LLM + 工具 + 记忆 + 规划。区别在于 Agent 能自主决策下一步行动，而不是被动响应。

### Q2: 说说你了解的 Agent 模式？

**参考答案**：
- ReAct：推理 + 行动交替
- CoT：思维链，逐步推理
- ToT：树状搜索多条路径
- Reflexion：加入自我反思

### Q3: Agent 如何保证输出稳定？

**参考答案**：
- 限制工具调用次数
- 添加输出格式约束
- 错误重试机制
- 人工审核节点

## 进阶题

### Q4: 如何设计一个多 Agent 协作系统？

**参考答案**：
- 明确 Agent 角色分工
- 设计 Agent 间通信协议
- 添加协调 Agent 调度任务
- 处理 Agent 冲突

### Q5: Agent 的安全风险有哪些？

**参考答案**：
- Prompt 注入
- 工具权限过大
- 敏感数据泄露
- 无限循环调用

### Q6: 如何评估 Agent 效果？

**参考答案**：
- 任务完成率
- 工具调用效率
- 错误恢复能力
- 用户满意度
