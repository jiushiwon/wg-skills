# VibeCoding 项目简历包装指南

一句话：把 VibeCoding 经历写成面试官认可的简历项目。

## 包装原则

1. **结果导向**：用数据说话
2. **技术深度**：体现排查问题和优化能力
3. **业务价值**：说明解决了什么问题

## 简历写法

### ❌ 原始写法
```
使用 AI 编程开发了一个商城小程序
```

### ✅ 包装后
```
AI 辅助开发全栈商城小程序（2 周 → 1 周）

- 使用 Cursor + Claude 开发小程序前后端
- 通过 AI 生成单元测试，覆盖率 80%
- 优化提示词模板，代码复用率提升 30%
- 技术栈：UniApp + Node.js + MySQL
```

## 项目描述模板

```
[项目名称] - [角色]

项目背景：[一句话描述]
技术方案：[使用的技术]
个人贡献：
- 负责 [模块 A]，通过 [方法] 解决 [问题]
- 负责 [模块 B]，优化 [指标] 从 X 提升到 Y
- 遇到的最大挑战：[问题]，解决方案：[方法]
```

## 常见问题回答

**Q: AI 编程算不算真本事？**
A: AI 编程考验的是需求分析、架构设计、问题排查能力，高效使用 AI 本身就是技术能力的体现。

**Q: 怎么证明代码是你写的？**
A: 展示代码审查记录、优化过程、技术决策文档，体现对代码的理解和控制力。
