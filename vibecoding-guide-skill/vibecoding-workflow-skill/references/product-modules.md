# 产品模块拆解参考

一句话：帮助用户快速拆解常见产品的功能模块。

## 电商系统

| 模块 | 核心功能 | 技术要点 |
|------|----------|----------|
| 用户系统 | 注册/登录/授权 | JWT、第三方登录 |
| 商品系统 | 商品管理/分类/搜索 | 搜索 ElasticSearch |
| 购物车 | 增删改查/持久化 | 状态管理 |
| 订单系统 | 下单/支付/状态流转 | 状态机、事务 |
| 支付系统 | 第三方支付接入 | 回调、幂等 |
| 库存系统 | 库存扣减 | 乐观锁、分布式锁 |
| 评价系统 | 评价/晒图 | 异步处理 |

## SaaS 系统

| 模块 | 核心功能 |
|------|----------|
| 多租户 | 租户隔离、数据权限 |
| RBAC | 角色权限、菜单权限 |
| 工作流 | 审批流、状态流转 |
| 消息中心 | 站内信、通知 |
| 报表系统 | 数据可视化 |

## 内容社区

| 模块 | 核心功能 |
|------|----------|
| 内容发布 | 富文本/图片/视频 |
| 内容推荐 | 标签、协同过滤 |
| 互动系统 | 点赞/评论/收藏 |
| 搜索系统 | 全文搜索 |
| 内容审核 | 敏感词、AI 审核 |
