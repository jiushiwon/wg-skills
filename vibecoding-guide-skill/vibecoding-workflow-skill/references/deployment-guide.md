# 部署指南

一句话：VibeCoding 场景下的部署选项与流程。

## 部署选项对比

| 平台 | 适合场景 | 特点 |
|------|----------|------|
| Vercel | 前端/全栈 | 免费额度大、部署快 |
| Netlify | 前端/全栈 | 功能丰富 |
| Railway | 全栈 | 容器化、灵活 |
| Render | 全栈 | 免费额度 |
| 阿里云/腾讯云 | 生产级 | 需配置、费用高 |

## 前端部署流程（Vercel 为例）

1. Git 推送代码
2. Vercel 自动检测框架
3. 配置环境变量
4. 自动部署

## 后端部署流程

1. 容器化（Docker）
2. 选择平台部署
3. 配置环境变量
4. 绑定域名（可选）

## 一键部署工具

- `superpowers:super-deploy-skills` - 一键部署套件
