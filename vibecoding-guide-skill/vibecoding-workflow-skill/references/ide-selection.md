# IDE 选型参考

一句话：帮助用户根据需求选择 AI 辅助编程 IDE。

## 主流 AI IDE 对比

| IDE | 厂商 | 核心特性 | 适合人群 |
|-----|------|----------|----------|
| Cursor | Anysphere | AI First、Tab 补全、Composer | 全栈开发者 |
| Windsurf | Codeium | Agent 模式、Flow | 想要"自动驾驶"体验 |
| VS Code + Claude Code | Anthropic | 可自定义 Skill、灵活 | 深度定制需求 |
| Zed | Zed Labs | 高性能、协作 | 追求极致性能 |

## 按需求选型

- **快速上手**：Cursor
- **想要 Agent 自动驾驶**：Windsurf
- **深度定制 / Skill 生态**：VS Code + Claude Code
- **性能优先 / Linux 开发者**：Zed
