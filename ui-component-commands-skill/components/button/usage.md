# button 使用示例

## 按钮类 (CSS 类)

### 主按钮

```html
<button class="btn-primary">确定</button>
<button class="btn-primary btn--small">小号</button>
<button class="btn-primary btn--large">大号</button>
<button class="btn-primary btn--block">块级按钮</button>
```

### 次按钮

```html
<button class="btn-secondary">取消</button>
```

### 幽灵按钮

```html
<button class="btn-ghost">详情</button>
```

### 图标按钮

```html
<button class="btn-icon">+</button>
<button class="btn-icon btn-icon--small">+</button>
<button class="btn-icon btn-icon--large">+</button>
```
