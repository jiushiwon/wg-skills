# combo 使用示例

## 组合类 (CSS 类)

### 图标标签

```html
<view class="icon-tag">
  <text class="icon-tag__icon">🔥</text>
  <text>热门</text>
</view>
```

### 徽章

```html
<!-- 带数字 -->
<view class="badge">
  <text>内容</text>
  <view class="badge__inner">5</view>
</view>

<!-- 红点 -->
<view class="badge badge--dot">
  <text>内容</text>
</view>
```
