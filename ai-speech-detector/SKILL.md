# AI Speech Detector - AI 风检测技能

本技能用于检测文案中的 AI 风味特征，包括开头套话、车轱辘话、虚词堆砌、短句短语、英文 AI 词汇、结构模式等，并提供修改建议。

## 检测维度

本技能从以下 8 个维度进行检测：

| 维度 | 优先级 | 检测内容 |
|------|--------|----------|
| **开头套话** | P0 | 说句大实话、扎心真相、揭秘 |
| **车轱辘话** | P0 | 首先/其次/最后、总而言之 |
| **虚词堆砌** | P1 | 其实、而且、但是、实际上 |
| **短句短语** | P1 | 列表式短句、两字短语堆砌 |
| **结尾套话** | P2 | 最后说一句、评论区见 |
| **英文 AI 词汇** | P1 | delve, tapestry, robust, pivotal 等 |
| **结构模式** | P1 | 二元对立、戏剧化断句、分形总结 |
| **格式模式** | P2 | 粗体首词列表、Unicode 箭头、破折号 |

## 风格校准（Voice Calibration）

本技能支持风格校准功能：用户提供自己的写作样本，AI 分析其风格特征后，用该风格重写文本。

### 使用方式

```
检测这段文案的 AI 风，并用我的风格重写。
这是我的写作样本：
[样本文本]
```

或从文件读取：

```
检测这段文案的 AI 风，参考我的写作风格。
我的写作样本在：path/to/my-writing.md
```

### 风格分析维度

1. **句子长度模式** - 短促型 / 流畅型 / 混合型
2. **词汇水平** - 口语化 / 学术型 / 中间型
3. **段落开头方式** - 直入型 / 铺垫型 / 反问型
4. **标点习惯** - 破折号 / 括号 / 分号 / 句号极简
5. **口头禅/重复语** - 填充词 / 转折词 / 强调词
6. **过渡方式** - 显式连接 / 隐式跳跃 / 场景转换

### 风格模仿规则

- **必须匹配**：句子长度、词汇等级、段落结构、标点偏好、过渡风格
- **可选增强**：个人观点、幽默感、情感表达

详见 [references/voice-calibration.md](references/voice-calibration.md)
