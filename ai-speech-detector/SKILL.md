# AI Speech Detector - AI 风检测技能

本技能用于检测文案中的 AI 风味特征，提供**量化评分**、**领域适配**、**风格校准**和**一键修复**，支持中文、英文和中英混合文本。

## 检测维度

### 核心维度（8个）

| 维度 | 优先级 | 权重 | 检测内容 |
|------|--------|------|----------|
| **开头套话** | P0 | 20% | 说句大实话、扎心真相、揭秘 |
| **车轱辘话** | P0 | 15% | 首先/其次/最后、总而言之 |
| **虚词堆砌** | P1 | 10% | 其实、而且、但是、实际上 |
| **短句短语** | P1 | 10% | 列表式短句、两字短语堆砌 |
| **结尾套话** | P2 | 5% | 最后说一句、评论区见 |
| **英文 AI 词汇** | P1 | 15% | delve, tapestry, robust, pivotal 等 |
| **结构模式** | P1 | 15% | 二元对立、戏剧化断句、分形总结 |
| **格式模式** | P2 | 5% | 粗体首词列表、Unicode 箭头、破折号 |

### 增强维度（3个）

| 维度 | 说明 |
|------|------|
| **领域适配** | 学术/社媒/电商/技术/公文领域的特色AI模式 |
| **节奏分析** | 句子长度分布、Burstiness 分数 |
| **中英混合** | 中英夹杂的AI表达模式 |

## AI 风指数算法

### 评分公式

```
AI风指数 = Σ(维度命中数 × 维度权重) / 文本总字数 × 100

维度得分 = (该维度命中数 / 该维度阈值) × 100，封顶100
```

### 阈值参考

| 维度 | 轻度阈值 | 中度阈值 | 重度阈值 |
|------|----------|----------|----------|
| 开头套话 | 1个 | 2个 | 3个+ |
| 车轱辘话 | 2个 | 4个 | 6个+ |
| 虚词堆砌 | 3个 | 6个 | 10个+ |
| 英文 AI 词汇 | 2个 | 5个 | 8个+ |
| 结构模式 | 1个 | 3个 | 5个+ |

### 综合评级

| 指数 | 评级 | 说明 |
|------|------|------|
| 0-30% | 🟢 自然 | 无明显AI痕迹 |
| 30-60% | 🟡 轻度 | 有少量AI模式，建议修改 |
| 60-80% | 🟠 中度 | AI风明显，需要修改 |
| 80%+ | 🔴 重度 | 高度疑似AI生成，必须重写 |

## 节奏分析（Burstiness）

### 检测指标

1. **句子长度变异系数**
   - 计算每句字数的标准差
   - AI 文本通常标准差 < 8
   - 人类文本通常标准差 > 15

2. **三句同长度检测**
   - 连续三句字数相差 ≤3
   - 出现一次 = 轻度
   - 出现三次+ = 重度

3. **段落句数一致性**
   - 每段句子数基本相同
   - 是 AI 的强信号

### Burstiness 分数

```
Burstiness = (句子长度标准差 - 平均句子长度) / (句子长度标准差 + 平均句子长度)

AI 文本：-0.5 ~ 0.2
人类文本：0.3 ~ 0.8
```

## 风格校准（Voice Calibration）

用户提供自己的写作样本，AI 分析其风格特征后，用该风格重写文本。

### 使用方式

```
检测这段文案的 AI 风，并用我的风格重写。
这是我的写作样本：
[样本文本]
```

### 风格分析维度

1. **句子长度模式** - 短促型 / 流畅型 / 混合型
2. **词汇水平** - 口语化 / 学术型 / 中间型
3. **段落开头方式** - 直入型 / 铺垫型 / 反问型
4. **标点习惯** - 破折号 / 括号 / 分号 / 句号极简
5. **口头禅/重复语** - 填充词 / 转折词 / 强调词
6. **过渡方式** - 显式连接 / 隐式跳跃 / 场景转换

详见 [references/voice-calibration.md](references/voice-calibration.md)

## 风格训练（Style Training）

用户提供一篇或多篇文章，AI 提炼风格特征，生成风格 Profile，然后用该风格创作全新内容。

### 与风格校准的区别

| 功能 | 风格校准 | 风格训练 |
|------|----------|----------|
| 样本数量 | 1 篇 | 1-3 篇（推荐 3 篇） |
| 输出 | 重写已有文本 | 创作全新内容 |
| 风格 Profile | 简要分析 | 详细的结构化风格报告 |
| 适用场景 | 修改已有文案 | 用个人风格写新文章 |

### 使用方式

**单样本训练：**

```
训练我的写作风格：
[粘贴一篇文章]

然后用这个风格写一篇关于[新主题]的文章。
```

**多样本训练（推荐）：**

```
训练我的写作风格，以下是 3 篇样本：

--- 样本 1 ---
[文章1]

--- 样本 2 ---
[文章2]

--- 样本 3 ---
[文章3]

然后用这个风格写一篇关于[新主题]的文章。
```

### 训练流程

```
用户提供样本（1-3篇）
    ↓
Step 1: 逐篇分析风格特征
Step 2: 提取跨样本共性，生成风格 Profile
Step 3: 输出风格特征报告 + 风格规则
Step 4: 用风格 Profile 创作新内容
    ↓
输出：风格特征报告 + 新文章
```

### 风格分析维度

1. **宏观结构** - 段落长度、段落组织、开头/结尾方式
2. **句子节奏** - 平均句长、句长分布、句式偏好、节奏模式
3. **词汇特征** - 词汇等级、用词偏好、高频词、修辞手法
4. **语气与视角** - 叙事视角、主观程度、情感温度、权威感
5. **标点与格式** - 标点偏好、段落间距、格式元素

### 输出格式

分析完成后输出：

```markdown
## 风格特征分析

### 宏观结构
- 段落长度：中等（150-300字）
- 段落组织：总分总结构
- 开头方式：场景引入
- 结尾方式：升华主题

### 句子节奏
- 平均句长：18字（中等）
- 节奏特征：快慢交替
- 句式偏好：陈述句为主（80%）

### 词汇特征
- 词汇等级：通俗偏口语
- 高频词："其实"、"说实话"、"你会发现"
- 修辞手法：善用比喻

### 语气与视角
- 叙事视角：第一人称"我"
- 主观程度：主观 opinionated
- 情感温度：热情中带幽默

### 标点与格式
- 标点偏好：破折号用于解释
- 段落间距：疏松

---

## 风格规则（创作时必须遵守）

1. 开头必须用场景引入
2. 段落长度控制在 150-300 字
3. 句子长短交替
4. 用"我"的视角，口语化表达
5. 善用比喻
6. 结尾要有升华
```
