# AI Speech Detector 🗣️

> 检测文案中的 AI 风味词汇、套话、车轱辘话，支持量化评分、领域适配、风格校准、一键修复和 Before/After 对比

## ✨ 功能特性

### 核心检测（8 个维度）

| 维度 | 说明 | 优先级 | 权重 |
|------|------|--------|------|
| **开头套话** | 说句大实话、扎心真相、揭秘 | P0 | 20% |
| **车轱辘话** | 首先/其次/最后、总而言之 | P0 | 15% |
| **虚词堆砌** | 其实、而且、但是、实际上 | P1 | 10% |
| **短句短语** | 列表式短句、两字短语堆砌 | P1 | 10% |
| **英文 AI 词汇** | delve, tapestry, robust, pivotal 等 | P1 | 15% |
| **结构模式** | 二元对立、戏剧化断句、分形总结 | P1 | 15% |
| **格式模式** | 粗体首词列表、Unicode 箭头、破折号 | P2 | 5% |
| **结尾套话** | 最后说一句、评论区见 | P2 | 5% |

### 增强检测（3 个维度）

| 维度 | 说明 |
|------|------|
| **领域适配** | 自动识别学术/社媒/电商/技术/公文领域的特色 AI 模式 |
| **节奏分析** | 句子长度分布、Burstiness 分数、三句同长度检测 |
| **中英混合** | 中英夹杂的 AI 表达模式检测 |

### 特色功能

| 功能 | 说明 |
|------|------|
| **AI 风指数** | 0-100% 量化评分，四级评级 |
| **一键修复** | 直接输出修复后的完整文本 |
| **Before/After 对比** | 逐段对比原文和修复后的文本 |
| **风格校准** | 提供写作样本，AI 模仿你的风格重写 |

## 🚀 快速开始

### 使用方式

当需要检测文案是否为 AI 生成、或审查文案质量时，告诉 Claude：

```
/ai-speech-detector
```

或描述性的自然语言：

```
帮我检测下这篇文章的AI风
看看这篇文章有没有AI味
分析下这段文案像不像AI写的
去除这段文字的AI味
```

### 指定文本

可以直接传入要检测的文案：

```
检测这段话的AI风：
近年来，随着XXX的快速发展，
我们需要首先了解其背景，
其次掌握其核心要点...
```

### 一键修复

直接获取修复后的文本：

```
检测这段文字并一键修复
去除这段文字的AI味并输出修复版
```

### 风格校准

提供你自己的写作样本，AI 会模仿你的风格重写：

```
检测这段文案的 AI 风，并用我的风格重写。
这是我的写作样本：
我其实不太确定这件事。你知道吗，有时候就是这样...
```

或从文件读取样本：

```
检测这段文案的 AI 风，参考我的写作风格。
我的写作样本在：path/to/my-writing.md
```

### 风格训练

提供 1-3 篇文章，AI 提炼你的风格 Profile，然后用该风格创作全新内容：

```
训练我的写作风格：
[粘贴一篇文章]

然后用这个风格写一篇关于[新主题]的文章。
```

多样本训练（效果更好）：

```
训练我的写作风格，以下是 3 篇样本：

--- 样本 1 ---
[文章1]

--- 样本 2 ---
[文章2]

--- 样本 3 ---
[文章3]

然后用这个风格写一篇关于[新主题]的文章。
```

**风格训练 vs 风格校准：**

| 功能 | 风格校准 | 风格训练 |
|------|----------|----------|
| 样本数量 | 1 篇 | 1-3 篇 |
| 输出 | 重写已有文本 | 创作全新内容 |
| 风格分析 | 简要 | 详细的结构化报告 |
| 适用场景 | 修改文案 | 用个人风格写新文章 |

## 📊 AI 风指数

### 评分标准

| 指数 | 评级 | 说明 | 建议 |
|------|------|------|------|
| 0-30% | 🟢 自然 | 无明显 AI 痕迹 | 无需修改 |
| 30-60% | 🟡 轻度 | 有少量 AI 模式 | 建议微调 |
| 60-80% | 🟠 中度 | AI 风明显 | 需要修改 |
| 80%+ | 🔴 重度 | 高度疑似 AI 生成 | 必须重写 |

### 指数计算

```
AI 风指数 = Σ(维度命中数 × 维度权重) / 文本总字数 × 100

各维度有独立阈值，超过阈值后分数封顶
```

## 📋 检测维度详解

### 开头套话（P0）

这些词通常出现在文章开头，故作神秘或激烈情绪：

| 词汇 | 风险 |
|------|------|
| 说句大实话 | 🔴 Critical |
| 扎心的真相 | 🔴 Critical |
| 揭秘 | 🔴 Critical |
| 说实在的 | 🟠 High |

### 车轱辘话（P0）

机械罗列要点，缺乏有机论述：

| 词汇 | 风险 |
|------|------|
| 首先 | 🔴 Critical |
| 其次 | 🔴 Critical |
| 最后 | 🔴 Critical |
| 总的来说 | 🔴 Critical |

### 英文 AI 词汇（P1）

| 类别 | 示例 |
|------|------|
| 重要性夸大词 | pivotal, crucial, vital, significant |
| AI 分析动词 | delve, embark, leverage, utilize |
| 诗意名词 | tapestry, landscape, realm, paradigm |
| 推销形容词 | vibrant, comprehensive, robust, seamless |

### 结构模式（P1）

| 模式 | 示例 |
|------|------|
| 二元对立 | Not a tool. A platform. |
| 戏剧化断句 | Speed. That's it. That's the tradeoff. |
| 自问自答 | The result? Devastating. |
| 三列滥用 | fast, reliable, scalable |
| 分形总结 | In this section, we'll explore... |

### 领域特色检测

自动识别文本领域，应用对应检测规则：

| 领域 | 特色 AI 模式 |
|------|-------------|
| **学术论文** | "The implications are significant", "highlighting the need for" |
| **社媒/短视频** | "家人们", "绝绝子", "沉浸式", "泰裤辣" |
| **电商文案** | "错过再等一年", "仅剩最后 X 件", "手慢无" |
| **技术博客** | "简单聊聊", "一文搞懂", "保姆级教程" |
| **公文/汇报** | "认真贯彻落实", "切实加强", "扎实推进" |

### 中英混合检测

| 模式 | 示例 |
|------|------|
| 英文词汇中式用法 | "这个 feature 真的很 robust" |
| 中英套话混合 | "我们需要先 align 一下" |
| 强行翻译腔 | "让我们 dive into 这个话题" |

### 节奏分析

检测句子长度分布，评估 Burstiness：

| 指标 | AI 文本 | 人类文本 |
|------|---------|----------|
| 句长标准差 | < 8 | > 15 |
| Burstiness | -0.5 ~ 0.2 | 0.3 ~ 0.8 |
| 三句同长度 | 频繁出现 | 很少出现 |

## ✏️ 修改技巧

### 1. 开头套话 → 删

```diff
- 说句大实话，这个行业真的卷不动了。
+ 这几年行业变化确实很大。
```

### 2. 车轱辘话 → 融

```diff
- 首先，要有目标。其次，要坚持。最后，要执行。
+ 关键是找准目标，然后持续行动。
```

### 3. 英文 AI 词汇 → 替换

```diff
- This is a robust solution that leverages cutting-edge technology.
+ This solution uses the latest technology and works reliably.
```

### 4. 结构模式 → 改写

```diff
- Not a tool. A platform.
+ This is a platform, not just a tool.
```

### 5. 风格校准 → 模仿你的风格

**你的样本：**
> 我其实不太确定这件事。你知道吗，有时候就是这样...

**AI 原文：**
> 综上所述，该方法具有显著的优越性...

**用你的风格重写：**
> 说实话，这个方法确实挺管用的。你知道吗...

## P0 问题（必须修改）

### 1. 开头套话 - "说句大实话"
位置：第 1 行

原文：
> 说句大实话，这个行业真的太难了。

问题：使用"说句大实话"故作神秘，属于典型的 AI 开头套路。

### Suggested fix
去掉套话，直接陈述：
> 近几年，行业竞争确实越来越激烈。
