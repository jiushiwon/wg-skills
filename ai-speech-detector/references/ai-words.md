## 七、中文模糊引用（P1）

| 表达 | 风险等级 |
|------|----------|
| "专家表示" | 🔴 Critical |
| "有人认为" | 🔴 Critical |
| "不少人觉得" | 🔴 Critical |
| "很多人不知道" | 🔴 Critical |
| "研究发现"（未指明研究） | 🔴 Critical |
| "数据显示"（未给出数据） | 🔴 Critical |
| "权威机构称" | 🔴 Critical |
| "业内人士透露" | 🔴 Critical |

### 修改建议

给出具体来源或删除。

```diff
- 专家表示，这个方法非常有效。
+ 张三（某大学教授）在2024年的研究中指出，这个方法使效率提升了30%。
```

## 八、中文二元对立模式（P1）

| 模式 | 风险等级 | 示例 |
|------|----------|------|
| "不是...而是..." | 🟠 High | 不是失败，而是学习 |
| "与其说...不如说..." | 🟠 High | 与其说是工具，不如说是平台 |
| "表面上看...实际上..." | 🟠 High | 表面上看是问题，实际上是机会 |

### 修改建议

直接陈述，不要对比。

```diff
- 这不是挑战，而是机遇。
+ 这个挑战背后隐藏着机遇。
```

## 修改技巧

### 1. 开头套话 → 删
```diff
- 说句大实话，这个行业真的卷不动了。
+ 这几年行业变化确实很大。
```

### 2. 车轱辘话 → 融
```diff
- 首先，要有目标。其次，要坚持。最后，要执行。
+ 关键是找准目标，然后持续行动。
```

### 3. 虚词堆砌 → 删
```diff
- 其实吧，我觉得这个问题呢，首先要搞清楚...
+ 这个问题的核心是...
```

### 4. 短句 → 长句
```diff
- 两个字：坚持靠谱。
+ 成功的本质是持续的投入和积累。
```

### 5. 英文 AI 词汇 → 替换
```diff
- This is a robust solution that leverages cutting-edge technology.
+ This solution uses the latest technology and works reliably.
```

### 6. 结构模式 → 改写
```diff
- Not a tool. A platform.
+ This is a platform, not just a tool.
```
