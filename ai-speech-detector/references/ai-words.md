## 修改技巧

### 1. 开头套话 → 删
```diff
- 说句大实话，这个行业真的卷不动了。
+ 这几年行业变化确实很大。
```

### 2. 车轱辘话 → 融
```diff
- 首先，要有目标。其次，要坚持。最后，要执行。
+ 关键是找准目标，然后持续行动。
```

### 3. 虚词堆砌 → 删
```diff
- 其实吧，我觉得这个问题呢，首先要搞清楚...
+ 这个问题的核心是...
```

### 4. 短句 → 长句
```diff
- 两个字：坚持靠谱。
+ 成功的本质是持续的投入和积累。
```
