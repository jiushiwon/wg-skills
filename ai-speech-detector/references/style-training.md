# 风格训练（Style Training）

本文档定义风格训练功能：用户提供一篇或多篇文章，AI 提炼风格特征，然后用该风格创作新内容。

---

## 使用方式

### 单样本训练

```
训练我的写作风格：
[粘贴一篇文章]

然后用这个风格写一篇关于[新主题]的文章。
```

### 多样本训练（推荐）

```
训练我的写作风格，以下是 3 篇样本：

--- 样本 1 ---
[文章1]

--- 样本 2 ---
[文章2]

--- 样本 3 ---
[文章3]

然后用这个风格写一篇关于[新主题]的文章。
```

### 从文件训练

```
训练我的写作风格，样本在以下文件：
- path/to/article1.md
- path/to/article2.md
- path/to/article3.md

然后用这个风格写一篇关于[新主题]的文章。
```

---

## 训练流程

```
用户提供的样本（1-3篇）
    ↓
Step 1: 风格分析
  - 逐篇分析每篇文章的风格特征
  - 找出跨样本的共性特征
  - 标记样本间的差异（如有矛盾，取多数特征）
    ↓
Step 2: 生成风格 Profile
  - 输出结构化的风格特征报告
  - 明确列出必须遵守的风格规则
    ↓
Step 3: 风格创作
  - 根据风格 Profile 和新主题创作
  - 严格遵守分析出的风格特征
    ↓
输出：风格特征报告 + 新文章
```

---

## 风格分析维度

### 1. 宏观结构

| 维度 | 分析内容 |
|------|----------|
| 段落长度 | 平均段长、段长变化范围 |
| 段落组织 | 总分总 / 线性推进 / 倒叙 / 碎片式 |
| 章节结构 | 是否有小标题、编号、分隔符 |
| 开头方式 | 开门见山 / 场景引入 / 设问 / 引用 |
| 结尾方式 | 总结 / 升华 / 留白 / 呼吁 |

### 2. 句子节奏

| 维度 | 分析内容 |
|------|----------|
| 平均句长 | 短（<10字）/ 中（10-25字）/ 长（>25字） |
| 句长分布 | 标准差、Burstiness 分数 |
| 句式偏好 | 陈述句 / 疑问句 / 感叹句 / 祈使句 比例 |
| 节奏模式 | 快慢交替 / 匀速 / 渐强渐弱 |

### 3. 词汇特征

| 维度 | 分析内容 |
|------|----------|
| 词汇等级 | 口语化 / 通俗 / 学术 / 混合 |
| 用词偏好 | 抽象词 vs 具体词、动词 vs 名词 |
| 高频词 | 口头禅、过渡词、情感词 |
| 修辞手法 | 比喻、排比、反问、反讽等 |

### 4. 语气与视角

| 维度 | 分析内容 |
|------|----------|
| 叙事视角 | 第一人称 / 第二人称 / 第三人称 |
| 主观程度 | 客观中立 / 主观 opinionated / 混合 |
| 情感温度 | 冷静 / 热情 / 幽默 / 讽刺 / 温暖 |
| 权威感 | 专家口吻 / 朋友聊天 / 观察者 |

### 5. 标点与格式

| 维度 | 分析内容 |
|------|----------|
| 标点偏好 | 破折号 / 括号 / 分号 / 省略号 |
| 段落间距 | 紧凑 / 疏松 |
| 格式元素 | 加粗 / 引用 / 列表 / 编号 |

## 风格 Profile 输出格式

分析完成后，输出结构化的风格 Profile：

```markdown
## 风格特征分析

### 宏观结构
- 段落长度：中等（150-300字），变化适中
- 段落组织：总分总结构
- 开头方式：场景引入，先描述一个画面
- 结尾方式：升华主题，带一点哲思

### 句子节奏
- 平均句长：18字（中等）
- 节奏特征：快慢交替，紧张情节用短句，抒情用长句
- 句式偏好：陈述句为主（80%），偶尔反问句增加互动感

### 词汇特征
- 词汇等级：通俗偏口语，偶尔用网络热词
- 高频词："其实"、"说实话"、"你会发现"
- 修辞手法：善用比喻，偶尔排比

### 语气与视角
- 叙事视角：第一人称"我"
- 主观程度：主观 opinionated，有明确观点
- 情感温度：热情中带一点幽默
- 权威感：朋友聊天式，不是专家口吻

### 标点与格式
- 标点偏好：破折号用于解释说明，括号用于补充
- 段落间距：疏松，每段之间有空行感

---

## 风格规则（创作时必须遵守）

1. 开头必须用场景引入，不要直接给结论
2. 段落长度控制在 150-300 字
3. 句子长短交替，不要连续三句同长度
4. 用"我"的视角，口语化表达
5. 善用比喻，但不要堆砌
6. 结尾要有升华，带一点哲思
7. 用破折号和括号做补充说明
8. 每段之间保持疏松感
```

---

## 多样本冲突处理

当多个样本风格不一致时：

| 冲突类型 | 处理方式 |
|----------|----------|
| 句长不一致（有的短有的长） | 取平均值，或说明风格跨度大 |
| 语气不一致（有的严肃有的幽默） | 询问用户偏好，或取样本最多的风格 |
| 视角不一致（有的第一人称有的第三人称） | 询问用户偏好 |
| 词汇等级不一致 | 取中间值，或标注为"混合风格" |

**建议**：提供同类型、同主题的样本，训练效果最佳。

---

## 创作时的风格保持检查

创作完成后，自我检查：

- [ ] 句长分布是否与样本一致？
- [ ] 开头方式是否符合样本风格？
- [ ] 高频词是否自然出现？
- [ ] 语气温度是否匹配？
- [ ] 标点习惯是否一致？
- [ ] 段落长度是否在范围内？

## 示例

### 用户提供的样本

> 凌晨三点，我又醒了。窗外的路灯透过窗帘缝照进来，在天花板上画出一道细长的光。我盯着那道光看了很久，忽然想起小时候奶奶说过，睡不着的时候数羊，数到一百只就能睡着。我试了，数到三百只，羊都跑累了，我还是醒着。
>
> 其实我也不知道自己在焦虑什么。工作还过得去，身体也没大问题，朋友圈看起来也挺正常。但就是有那种说不清的空虚感，像鞋子里进了一粒沙子，走每一步都不舒服，但脱下来又找不到那粒沙子在哪里。

### 风格 Profile

**宏观结构**：
- 段落长度：短（100-200字）
- 开头方式：场景引入（时间+动作）
- 结尾方式：留白，用比喻收束

**句子节奏**：
- 平均句长：22字（中等偏长）
- 节奏：舒缓，带有停顿感
- 句式：陈述句为主，偶尔用"其实我也不知道"打破节奏

**词汇特征**：
- 词汇等级：口语化，生活化
- 高频词："其实"、"就是"、"那种"
- 修辞：善用比喻（"像鞋子里进了一粒沙子"）

**语气与视角**：
- 第一人称"我"
- 主观、内省、略带忧郁但不沉重
- 朋友聊天式

**标点**：
- 逗号多，句号少，有绵长感
- 偶尔用破折号

### 风格规则

1. 开头用具体时间+具体动作引入场景
2. 段落短，100-200字
3. 句子中等偏长，带停顿感
4. 用生活化比喻表达抽象情感
5. 第一人称，内省式语气
6. 结尾留白，用比喻收束
7. 不要给明确结论，保持模糊感

### 用这个风格创作

**主题**：关于独处

> 周末的下午，我没有出门。窗帘拉着，屋里很暗，只有电脑屏幕发出一点蓝光。我坐在椅子上，没有开灯，也没有开音乐，就这样坐着。
>
> 其实独处并不可怕。可怕的是你发现自己竟然习惯了这种安静，甚至有点享受。就像一个人走了很远的路，突然停下来，发现四周空无一人，没有声音，没有期待，只有自己。那种感觉说不清是自由还是孤独，或者两者本来就是一回事。
