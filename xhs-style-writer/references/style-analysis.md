# 风格档案提取指南

逐篇笔记按以下 12 个维度提取。**铁律：每条结论后必须附一句原文例句。** 没有例句支撑的特征不写进档案——那是猜测，不是风格。

## 12 个提取维度

### 1. 人称与自称
- 第一人称用什么：我 / 本人 / 咱 / 姐妹 / 博主自称（如"小 A"）
- 对读者的称呼：你 / 姐妹们 / 宝子 / 家人们 / 直呼"你"

### 2. 语气基调
- 归为以下之一或组合：闺蜜聊天 / 老师授课 / 吐槽自嘲 / 冷静干货 / 打鸡血 / 示弱求助
- 记录情绪浓度：平铺直叙 ←→ 感叹号连发

### 3. 口头禅与高频词
- 开头常用词（如"说真的""谁懂啊""先上结论"）
- 转折常用词（如"但是""结果呢""重点来了"）
- 结尾常用词（如"就这样""懂的都懂""冲"）
- 个人专属梗或固定句式（如每篇必出现的自嘲）

### 4. 句式与节奏
- 平均句长：碎句为主（≤15 字）/ 长短交错 / 长句铺陈
- 是否用单字成段（"绝了。""真的。""冲。"）
- 排比使用频率：从不 / 偶尔 / 频繁

### 5. emoji 习惯
- 频率：每段必有 / 只在标题 / 几乎不用
- 位置：句首 / 句尾 / 替代标点
- 偏好的具体 emoji（列出 3-5 个高频的，如 ✨📌💡）
- 是否用符号替代 emoji（如 ❶❷❸、▪️、👉）

### 6. 标点习惯
- 感叹号/波浪号/省略号的使用密度
- 是否用"～"结尾表语气
- 括号用法：补充说明 / 吐槽弹幕式（如"（别问，问就是踩过坑）"）

### 7. 段落与排版
- 每段行数：1 行 / 2-3 行 / 大段落
- 段间是否空行
- 是否用分隔符（——、·、✦）
- 是否有固定的板块结构（如"先说结论→再讲过程→最后避坑"）

### 8. 标题套路
- 标题结构偏好：数字式 / 提问式 / 身份式 / 反差式（对照 `title-formulas.md` 归类）
- 标题 emoji 位置与数量
- 标题里是否出现人称（"我""30 岁宝妈"）

### 9. 开头钩子类型
- 偏好哪种开头：痛点共鸣 / 结果前置 / 场景代入 / 反常识 / 直接提问（对照 `note-structure.md`）
- 开头几句话进入正题

### 10. 结尾方式
- 互动提问 / 引导收藏关注 / 预告下期 / 戛然而止
- 是否有固定结尾签名句

### 11. 细节密度
- 具体数字、品牌名、价格、地点出现的频率
- 个人经历的讲述方式：流水账 / 关键场景特写 / 只给结论

### 12. 话题标签习惯
- 标签数量、位置（文末/穿插）
- 大词与精准词的配比（如 #护肤 vs #油敏皮早C晚A）

## 风格档案输出模板

分析完所有样本后，汇总为以下结构（保存为 `xhs-style-<账号名>.md` 供复用）：

```markdown
# 风格档案：<账号名或用户名>
> 样本：<N> 篇笔记 | 建档日期：<YYYY-MM-DD> | 置信度：高/中/低（样本 ≥5 篇为高）

## 一句话画像
<例：30 岁沪漂宝妈的闺蜜式吐槽，干货里夹自嘲，emoji 克制但感叹号很多>

## 语言特征
- 自称：<我>，称呼读者：<姐妹们>
- 语气：<闺蜜聊天 + 偶尔自嘲>
- 口头禅：开头"<说真的>"；转折"<但说实话>"；结尾"<懂的都懂>"
- 句式：<碎句为主，平均每句 12 字，常用单字成段>
- 标点：<感叹号每篇 5-8 个，爱用（括号吐槽）>

## emoji 与排版
- emoji：<每段 1 个，固定用 ✨📌❗，只放句尾>
- 分段：<每段 1-2 行，段间空行，用 ❶❷❸ 列点>
- 结构：<结论前置 → 过程 → 避坑清单>

## 标题与开头
- 标题套路：<数字+身份式，如"30 岁宝妈|3 个月瘦 12 斤的真实记录"，emoji 在开头>
- 开头钩子：<痛点共鸣式，第一句直击场景>

## 结尾与标签
- 结尾：<互动提问式，固定句"你们有同款困扰吗👇">
- 标签：<6-7 个，2 大词 + 4 精准词，放文末>

## 禁用项（本人从不做的）
- <例：从不用"绝绝子"、从不写超过 3 行的段落、从不堆砌排比>

## 代表例句（写作时的语气锚点）
1. "<原文例句 1>"
2. "<原文例句 2>"
3. "<原文例句 3>"
```

## 置信度规则

- ≥5 篇样本：高置信度，所有维度可下结论。
- 3-4 篇：中置信度，"禁用项"只写确定出现的，其余标注"样本不足待验证"。
- <3 篇：低置信度，只出"一句话画像 + 语言特征"，明确告知用户档案偏薄。
