# 登录页模板

> PC端React登录页，3套风格可选

## 目录结构

```
login/
├── style-1-simple.tsx + css   # 风格1：简约现代
├── style-2-vibrant.tsx + css # 风格2：活力炫彩
└── style-3-dark.tsx + css    # 风格3：深色科技
```

## 风格预览

### 风格1：简约现代
- 白色背景 (#F5F7FA)
- 蓝色主调 (#2563EB)
- 适合企业后台

### 风格2：活力炫彩
- 渐变背景
- 橙色主调 (#FF6B6B)
- 适合电商/社交

### 风格3：深色科技
- 深蓝背景 (#0A0E27)
- 霓虹青色 (#00D9FF)
- 适合科技产品
