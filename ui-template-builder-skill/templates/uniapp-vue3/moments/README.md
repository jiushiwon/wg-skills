# Moments 朋友圈

uniapp 朋友圈/动态页面模板。

## 风格

| 风格 | 文件 | 特点 |
|------|------|------|
| 默认 | style-1-default.vue | 通用风格，图片网格 |
| 微信 | style-2-wechat.vue | 仿微信朋友圈 |
| 小红书 | style-3-card.vue | 瀑布流卡片风格 |

## 使用

```vue
<template>
  <moments-page />
</template>

<script setup>
import Moments from './moments/style-1-default.vue'
</script>
```
