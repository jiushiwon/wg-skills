# 商品详情页模板

> uniapp商品详情页，4套风格

## 目录结构

```
product-detail/
├── style-1-simple.vue   # 风格1：简约现代
├── style-2-vibrant.vue  # 风格2：活力炫彩
├── style-3-premium.vue  # 风格3：高端商务
└── style-4-nature.vue   # 风格4：清新自然
```

## 包含功能

- 轮播图（swiper）
- 价格区域（现价+原价+标签）
- 规格选择入口
- 商品详情区
- 评价晒单区
- 相关推荐区
- 底部操作栏（客服+收藏+购物车+购买）

## 风格说明

### 风格1：简约现代
- 白色卡片
- 橙色价格
- 简洁布局

### 风格2：活力炫彩
- 粉橙渐变按钮
- 大圆角卡片
- 活泼氛围

### 风格3：高端商务
- 深色背景
- 金色点缀
- 精致感

### 风格4：清新自然
- 绿色主题
- 浅绿背景
- 圆润卡片
