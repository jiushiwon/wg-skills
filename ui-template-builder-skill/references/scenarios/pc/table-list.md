# 表格列表页模板

## 页面概述

后台管理系统中最常用的数据展示页面，支持筛选、排序、分页、批量操作。

## 适用场景

- 用户列表
- 订单管理
- 商品管理
- 数据报表

## 模板结构

```
table-list/
├── page-header.vue     # 页面标题
├── search-bar.vue     # 搜索栏
├── filter-bar.vue     # 筛选栏
├── table.vue          # 表格组件
├── table-column.vue   # 列配置
├── pagination.vue     # 分页
├── action-bar.vue     # 操作栏
└── mock.json
```

## 布局结构

```
┌──────────────────────────────────────────┐
│  用户管理                                │
├──────────────────────────────────────────┤
│  [搜索框]  [搜索]  [+ 新增]  [导出]     │
├──────────────────────────────────────────┤
│  状态: [全部▼]  时间: [范围选择器]      │
├──────────────────────────────────────────┤
│ ◉ │ ID │ 姓名 │ 手机 │ 状态 │ 操作     │
│ ──┼────┼──────┼───────┼──────┼────────  │
│ ○ │ 001│ 张三 │ 138.. │ 正常 │ [查看][编辑][删除] │
│ ○ │ 002│ 李四 │ 139.. │ 禁用 │ [查看][编辑][删除] │
├──────────────────────────────────────────┤
│  共 100 条  [< 1 2 3 ... 10 >]  每页 10 │
└──────────────────────────────────────────┘
```

## 区域详解

### 1. 页面标题

**内容**：
- 标题
- 描述（可选）

### 2. 搜索栏

**内容**：
- 关键词输入
- 搜索按钮
- 重置按钮

**高级搜索**：
- 更多筛选条件
- 展开/收起

### 3. 筛选栏

**内容**：
- 下拉选择
- 时间范围
- 状态筛选

### 4. 操作栏

**内容**：
- 新增按钮
- 批量操作
- 导出/导入
- 自定义操作

### 5. 表格

**列类型**：
- 文本
- 图片
- 标签
- 开关
- 按钮
- 自定义

**功能**：
- 排序
- 多选
- 斑马纹
- 边框

### 6. 分页

**内容**：
- 总数
- 当前页
- 每页条数
- 页码跳转

## 布局变体

### 变体1：紧凑高效

**特点**：
- 行高 32-40px
- 密集信息展示
- 适合大数据量

### 变体2：宽敞舒适

**特点**：
- 行高 48-56px
- 操作按钮明显
- 适合需要频繁操作的场景

### 变体3：卡片表格

```
┌──────────────────────────────────────────┐
│  ┌──────────────────────────────────┐   │
│  │  姓名：张三          状态：正常   │   │
│  │  手机：138****8888                │   │
│  │  [查看] [编辑] [删除]             │   │
│  └──────────────────────────────────┘   │
│  ┌──────────────────────────────────┐   │
│  │  姓名：李四          状态：禁用   │   │
│  │  手机：139****9999                │   │
│  │  [查看] [编辑] [删除]             │   │
│  └──────────────────────────────────┘   │
└──────────────────────────────────────────┘
```

**特点**：
- 卡片式行
- 可展开详情
- 适合需要展示大量字段的场景

## 表格功能

### 排序

- 单列排序
- 多列排序
- 排序图标

### 筛选

- 筛选项配置
- 筛选状态记录
- 清空筛选

### 分页

- 前端分页
- 后端分页
- 每页条数选择

### 行操作

- 查看
- 编辑
- 删除
- 自定义操作

### 批量操作

- 批量删除
- 批量导出
- 批量修改状态
