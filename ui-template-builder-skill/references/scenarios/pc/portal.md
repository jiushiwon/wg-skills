# 门户网站模板

## 页面概述

企业/产品官方网站首页，展示品牌形象、产品服务、案例资讯。

## 适用场景

- 企业官网
- 产品官网
- 营销落地页
- 个人主页

## 模板结构

```
portal/
├── header.vue         # 顶部导航
├── hero.vue          # 首屏区域
├── features.vue      # 核心优势
├── products.vue      # 产品展示
├── cases.vue         # 案例展示
├── about.vue         # 关于我们
├── partners.vue      # 合作伙伴
├── footer.vue        # 底部信息
└── mock.json
```

## 区域详解

### 1. 顶部导航

**内容**：
- Logo
- 导航链接
- CTA按钮

**交互**：
- 滚动后背景变化
- 移动端 hamburger 菜单

### 2. 首屏 Hero

**内容**：
- 大标题
- 副标题
- CTA按钮
- 背景图/视频

**高度**：80-100vh

### 3. 核心优势

**内容**：
- 图标 + 标题 + 描述
- 3-6个特点

### 4. 产品展示

**内容**：
- 产品图片
- 产品名称
- 简要描述
- 了解更多链接

### 5. 客户案例

**内容**：
- 案例图片
- 客户名称
- 简短评价

### 6. 合作伙伴

**内容**：
- 客户/合作伙伴Logo

### 7. 底部

**内容**：
- 版权信息
- 联系方式
- 社交媒体
- 备案号
