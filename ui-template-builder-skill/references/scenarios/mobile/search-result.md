# 搜索结果页模板

## 页面概述

展示搜索结果，支持筛选、排序、分页。

## 适用场景

- 商品搜索
- 内容搜索
- 用户搜索

## 模板结构

```
search-result/
├── search-bar.vue      # 搜索栏
├── filter-bar.vue     # 筛选栏
├── result-list.vue    # 结果列表
├── result-item.vue    # 结果项
├── empty-state.vue    # 空结果
└── mock.json
```

## 布局结构

```
┌─────────────────────────────┐
│  < 返回  [ 输入框 ]  🔍    │
├─────────────────────────────┤
│  搜索"手机"                │
│  找到 1000+ 个结果         │
├─────────────────────────────┤
│ [综合] [销量] [价格]       │
│ [品牌▼] [价格区间▼]       │
├─────────────────────────────┤
│ ┌────┐ 商品名称           │
│ │图片│ ¥99.00            │
│ └────┘ 已售1000件         │
├─────────────────────────────┤
│ ┌────┐ 商品名称           │
│ │图片│ ¥99.00            │
│ └────┘ 已售1000件         │
└─────────────────────────────┘
```

## 区域详解

### 1. 搜索栏

**内容**：
- 返回按钮
- 输入框（保留搜索词）
- 搜索按钮

### 2. 搜索信息

**内容**：
- 搜索关键词
- 结果数量

### 3. 筛选栏

**内容**：
- 排序：综合/销量/价格/新品
- 筛选：品牌/价格区间/属性

### 4. 结果列表

**布局**：同商品列表
