# 首页模板

## 页面概述

App入口页面，展示核心功能入口、运营活动、推荐内容。

## 适用场景

- 电商首页
- 资讯首页
- 服务入口首页

## 模板结构

```
home/
├── search-bar.vue      # 搜索栏
├── banner.vue          # 轮播图
├── nav-grid.vue        # 功能导航
├── notice.vue          # 公告栏
├── section-title.vue   # 区块标题
├── product-list.vue   # 商品推荐
├── article-list.vue   # 文章推荐
└── mock.json
```

## 区域详解

### 1. 搜索栏

**位置**：顶部固定

**内容**：
- 搜索图标
- 搜索输入框（可点击跳转）
- 扫码/消息图标（可选）

**交互**：
- 点击跳转搜索页

### 2. 轮播图 Banner

**位置**：搜索栏下方

**规格**：
- 高度：150-200px
- 间距：左右12px
- 圆角：12px
- 指示点：底部居中

**内容**：
- 运营活动图
- 跳转链接

### 3. 功能导航

**位置**：Banner下方

**布局**：4-8个网格

**内容**：
- 图标
- 文字
- 典型：新品、分类、直播、客服、订单、会员

### 4. 公告栏

**位置**：导航下方

**内容**：
- 公告图标
- 滚动公告内容

**交互**：
- 点击跳转公告详情

### 5. 推荐区域

**位置**：公告下方

**内容**：
- 标题
- 商品/内容列表

**布局**：
- 网格（2-3列）
- 横向滑动
- 列表

### 6. 活动区域

**位置**：推荐下方

**内容**：
- 大图活动
- 拼团/秒杀入口

## 交互规范

1. **下拉刷新**：刷新首页内容
2. **上拉加载**：加载更多推荐
3. **滚动吸顶**：搜索栏滚动时吸顶
4. **图片懒加载**：优化性能
