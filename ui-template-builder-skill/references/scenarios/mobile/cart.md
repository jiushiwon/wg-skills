# 购物车页面模板

## 页面概述

展示用户购物车中的商品，支持商品选择、数量编辑、删除等操作。

## 适用场景

- 电商购物车
- 点餐购物车
- 服务预约车

## 模板结构

```
cart/
├── cart-header.vue     # 顶部全选
├── cart-list.vue      # 购物车商品列表
├── cart-item.vue      # 单个商品项
├── cart-empty.vue     # 空购物车
├── cart-bottom.vue    # 底部结算栏
├── coupon-panel.vue   # 优惠券面板
└── mock.json
```

## 区域详解

### 1. 顶部操作栏

**内容**：
- 页面标题
- 编辑按钮（显示/隐藏多选框）

**交互**：
- 点击编辑切换多选模式

### 2. 购物车列表

**按店铺分组**：
- 店铺名称
- 店铺内商品列表

**商品项内容**：
- 选择框（复选框）
- 商品图片
- 商品名称
- 规格/属性
- 单价
- 数量编辑（+/-按钮）
- 删除按钮

### 3. 空购物车

**内容**：
- 购物车图标
- 提示文案："购物车还是空的"
- 去逛逛按钮

### 4. 底部结算栏

**内容**：
- 全选框
- 已选数量
- 合计金额
- 结算按钮

**高度**：50px + safe-area

**交互**：
- 点击结算跳转订单确认页

## 交互规范

1. **底部固定**：结算栏固定底部
2. **安全区域**：适配 safe-area
3. **数量限制**：最低1，最高库存
4. **滑动删除**：左滑删除商品
5. **选中计算**：只计算已选商品
6. **全选逻辑**：店铺全选时自动选中

## 特殊状态

### 无库存

- 灰态显示
- 提示"已售罄"
- 不可选中

### 已降价

- 显示原价划线
- 提示"降价X元"

### 限购

- 提示"限购1件"
- 数量锁定为1
