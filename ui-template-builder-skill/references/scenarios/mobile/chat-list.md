# 聊天列表页模板

## 页面概述

展示会话列表，支持未读消息、置顶、搜索等功能。

## 适用场景

- 消息中心
- 客服列表
- 私信列表
- 社区消息

## 模板结构

```
chat-list/
├── search-bar.vue      # 搜索栏
├── tab-bar.vue        # 消息分类
├── chat-item.vue      # 会话项
├── conversation.vue   # 聊天详情（另页）
├── unread-badge.vue   # 未读角标
└── mock.json
```

## 区域详解

### 1. 顶部操作

**内容**：
- 页面标题
- 搜索框

### 2. 消息分类

**位置**：搜索下方

**类型**：
- 全部
- 客服
- 订单
- 系统
- 互动

### 3. 会话列表

**会话项内容**：
- 头像
- 名称/标题
- 最后一条消息
- 时间
- 未读数角标
- 置顶标识

**布局**：
```
┌────┐  名称              时间
│头像│  最后一条消息...    未读数
└────┘
```

### 4. 会话状态

| 状态 | 显示 |
|------|------|
| 未读 | 粗体文字 + 角标 |
| 已读 | 正常文字 |
| 置顶 | 置顶图标 |
| 草稿 | 草稿图标 |
| 免打扰 | 关闭图标 |

## 布局变体

### 变体1：标准列表

**特点**：
- 头像 + 名称 + 消息 + 时间
- 未读角标
- 适合大多数场景

### 变体2：带分组

```
┌─────────────────────────────┐
│  今天                       │
├─────────────────────────────┤
│ ┌────┐  客服               │
│ └────┘                     │
├─────────────────────────────┤
│  昨天                       │
├─────────────────────────────┤
│ ┌────┐  朋友A              │
│ └────┘                     │
├─────────────────────────────┤
│  更早                       │
├─────────────────────────────┤
│ ┌────┐  朋友B              │
│ └────┘                     │
└─────────────────────────────┘
```

**特点**：
- 按时间分组
- 今天/昨天/更早
- 适合消息较多的场景

### 变体3：带图片预览

```
┌─────────────────────────────┐
│ ┌────┐                     │
│ │头像│  [图片]              │
│ │    │  10:30              │
│ └────┘                     │
└─────────────────────────────┘
```

**特点**：
- 最后消息为图片时显示缩略图
- 适合社交/社区

## 交互规范

1. **点击跳转**：点击会话跳转到聊天详情
2. **左滑操作**：删除/标记已读
3. **长按操作**：置顶/删除/免打扰
4. **下拉刷新**：刷新消息列表
