# 订单列表页模板

## 页面概述

展示用户订单列表，支持按状态筛选、查看订单详情。

## 适用场景

- 电商订单列表
- 预约订单列表
- 会员订单记录

## 模板结构

```
order-list/
├── filter-tabs.vue    # 状态筛选Tabs
├── order-card.vue     # 订单卡片
├── order-item.vue     # 订单内商品项
├── order-empty.vue    # 空状态
└── mock.json
```

## 布局结构

## 状态筛选

### 订单状态

| 状态 | 说明 | 按钮操作 |
|------|------|----------|
| 全部 | 所有订单 | - |
| 待付款 | 未支付 | 去支付 |
| 待发货 | 已支付未发货 | 提醒发货 |
| 待收货 | 已发货 | 查看物流/确认收货 |
| 已完成 | 已确认 | 再次购买/删除 |
| 售后 | 退款/退货中 | 查看售后 |

## 布局变体

### 变体1：标签筛选

**特点**：
- 顶部固定Tabs
- 卡片式订单展示
- 操作按钮在卡片底部
- 适合操作较多的订单

### 变体2：时间线

```
┌─────────────────────────────┐
│         2024年1月            │
├─────────────────────────────┤
│  ● 已完成    01-15 14:30   │
│  │ 确认收货                 │
│  │                         │
│  ├ 已发货    01-14 09:00   │
│  │ 顺丰速运 SF123456789    │
│  │                         │
│  ├ 已支付    01-13 10:00   │
│  │                         │
│  └ 已下单    01-13 09:55   │
└─────────────────────────────┘
```

**特点**：
- 按时间倒序
- 纵向时间轴展示
- 适合物流跟踪

## 订单卡片内容

### 头部

- 店铺名称
- 订单状态
- 订单号（可选）

### 商品区

- 商品图片
- 商品名称
- 规格（可选）
- 单价 × 数量

### 底部

- 商品件数
- 合计金额
- 操作按钮

## 操作按钮

### 待付款

- 查看订单
- 去支付
- 取消订单

### 待发货

- 查看订单
- 提醒发货
- 申请退款

### 待收货

- 查看订单
- 查看物流
- 确认收货

### 已完成

- 查看订单
- 再次购买
- 删除订单
- 申请售后

## 交互规范

1. **Tabs吸顶**：筛选栏滚动时吸顶
2. **卡片点击**：点击卡片跳转详情
3. **按钮点击**：点击按钮执行对应操作
4. **下拉刷新**：刷新订单列表
5. **上拉加载**：加载更多订单
