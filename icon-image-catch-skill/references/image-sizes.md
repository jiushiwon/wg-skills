# 项目图片尺寸参考规范

本文件汇总常见项目场景下的图片尺寸与比例建议，供 `image-catch-skill` / `icon-catch-skill` 抓图时参考。尺寸以设计稿 1x 为准，实际导出时按项目要求输出 2x/3x。

## 通用比例速查

| 比例 | 常见用途 | 示例尺寸 |
|---|---|---|
| 1:1 | 头像、商品主图、图标包装 | 64×64 / 200×200 / 400×400 |
| 4:3 | 文章封面、卡片配图 | 400×300 / 800×600 |
| 16:9 | Banner、视频封面、卡片图 | 750×422 / 1920×1080 |
| 3:4 | 人物写真、竖版封面 | 600×800 / 750×1000 |
| 2:1 | 横幅 banner | 750×375 / 1200×600 |
| 21:9 | 超宽 banner | 1920×823 |
| 9:16 | 手机全屏背景、短视频封面 | 750×1334 / 1080×1920 |

## 抓取时的建议

1. **先定比例，再定尺寸**：告诉 `image-catch` `--width` 时，同时考虑目标比例。
2. **宁可大不要小**：抓图时选比目标稍大的尺寸，本地再用 `image-forge-skill` 裁剪/压缩。
3. **图标只用一个尺寸链**：同一项目所有图标保持同一画布尺寸，避免视觉不统一。
4. **移动端优先 2x/3x 素材**：设计稿 750px 宽对应 2x，实际开发可用 `rpx` 或 `vw` 自适应。

## 推荐默认配置

| 目标 | 推荐命令 |
|---|---|
| 手机 Banner | `--width 750 --orientation landscape` |
| 手机卡片图 | `--width 686 --height 386 --fit cover` |
| 手机头像 | `--width 240 --height 240 --fit cover` |
| PC Hero | `--width 1920 --height 600 --fit cover` |
| PC 卡片封面 | `--width 800 --height 450 --fit cover` |
| 人物写真 | `--width 800 --height 1000 --fit cover` |
| 方形商品图 | `--width 800 --height 800 --fit cover` |
