# 图标库

本文件定义节点可用的图标。每个图标用简单 SVG path / shape 表示，保证在不同风格下都能快速渲染。

## 图标名与语义

| 图标名 | 语义 | SVG 实现要点 |
|--------|------|--------------|
| `user` | 用户/输入 | 圆形头像 + 身体 |
| `brain` | AI/思考/规划 | 大脑轮廓 |
| `code` | 代码/生成 | `</>` 符号 |
| `play` | 运行/预览 | 三角形播放按钮 |
| `eye` | 检查/验收 | 眼睛形状 |
| `loop` | 循环/反馈 | 弯曲箭头 |
| `chat` | 对话/反馈 | 对话框 |
| `git` | 版本控制 | 分支节点 |
| `server` | 服务器/环境 | 服务器机箱 |
| `rocket` | 发布/上线 | 火箭 |
| `database` | 数据库 | 圆柱体 |
| `search` | 检索 | 放大镜 |
| `document` | 文档 | 文件页角 |
| `check` | 完成 | 对勾 |

## 风格适配

- **flat**：用填充 shape，颜色与节点语义匹配
- **sketchy**：用简单线条，保留手绘感
- **dark**：用浅色线条/填充，确保在深色背景可见

## 图标尺寸

- 图标区域：32×32px
- 圆形背景：r=16
- 图标内容：约 20×20px，居中

## 实现约定

在 `scripts/generate_diagram.py` 中，每个图标用一个函数返回 SVG path 列表：

```python
def icon_user(cx, cy, stroke, fill):
    return [
        f'<path d="..." fill="{stroke}"/>',
    ]
```

`cx`、`cy` 为图标中心坐标，`stroke` 为图标主色，`fill` 为背景色。
