# Sketchy 手绘草图风格规范

适合公众号、视频、演讲。模仿 Excalidraw / Napkin 的手绘质感。

## 视觉特征

- 边框带有轻微随机抖动，不完全直线
- 圆角不规则
- 图标为简笔画风格
- 箭头带手绘弯曲感
- 整体配色柔和，但保留语义色

## 颜色

```
背景:       #fffef8（暖白）
容器背景:   #fffdf5
容器边框:   #d4c5b0
节点填充:   #fffef8
节点边框:   #8b7e66（铅笔灰）
主文字:     #3d3d3d
副文字:     #7a7a7a
主流程箭头: #4a7c59（墨绿）
迭代箭头:   #a35c5c（砖红）
```

## 边框绘制

使用 rough.js 风格：
- 用多段短直线模拟曲线
- 路径点带有 ±2px 随机偏移
- 圆角用 4-6 段折线代替

## 图标

- 简笔画：用户是圆圈加两条线，代码是 </>，火箭是三角形加火焰
- 不要细节，保持像素级简洁

## 箭头

- 线条略微弯曲，不是完全直线
- 箭头头部为手绘三角形
- 迭代循环用虚线或手绘波浪线

## 字体

- 使用手写风格字体栈：'Comic Sans MS', 'Chalkboard SE', 'Marker Felt', 'PingFang SC', sans-serif
- 标题 16px，副标题 13px
