# Sketchy 手绘草图风格规范

适合公众号、视频、演讲。模拟手账/便签的轻松质感。

## 视觉特征

- 暖白背景（#fffef8），柔和不刺眼
- 节点边框为铅笔灰（#8b7e66），营造手绘线稿感
- 图标为简笔画风格，语义清晰但不精细
- 箭头为简洁直线，配色保留语义色
- 使用手写风格字体栈

## 颜色

```
背景:       #fffef8（暖白）
容器背景:   #fffdf5
容器边框:   #d4c5b0
节点填充:   #fffef8
节点边框:   #8b7e66（铅笔灰）
主文字:     #3d3d3d
副文字:     #7a7a7a
主流程箭头: #4a7c59（墨绿）
迭代箭头:   #a35c5c（砖红）
```

## 节点样式

- 圆角矩形，rx=8, ry=8
- 描边宽度 1.5px，铅笔灰
- 左侧圆形图标背景，简笔画图标

## 图标

- 简笔画：用户是圆圈加两条线，代码是 </>，火箭是三角形加火焰
- 不要细节，保持像素级简洁

## 箭头

- 简洁直线
- 箭头头部为标准三角形
- 迭代循环用虚线

## 字体

- 使用手写风格字体栈：'Comic Sans MS', 'Chalkboard SE', 'Marker Felt', 'PingFang SC', sans-serif
- 标题 15px，副标题 12px

## 说明

当前实现通过配色与字体模拟手绘氛围，未引入 rough.js 的物理抖动算法。如需更强烈的粗糙边框效果，可在后续版本中扩展 `scripts/generate_diagram.py` 的 sketchy 渲染分支。
