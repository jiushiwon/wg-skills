# Dark 风格规范

适合技术博客、GitHub README、深色主题展示。

## 颜色

```
背景:       #0f0f1a
容器背景:   #1a1a2e
容器边框:   #2d2d44
节点填充:   #16162a
节点边框:   #3f3f5f
主文字:     #e4e4f0
副文字:     #9ca3af
主流程箭头: #60a5fa
迭代箭头:   #c084fc
```

## 节点样式

- 深色填充 + 浅色边框
- 图标背景用半透明色：rgba(96, 165, 250, 0.15)
- 标题 15px 加粗白色

## 箭头

- 亮蓝色主流程箭头
- 亮紫色迭代箭头
- 可加轻微发光效果（filter: drop-shadow）

## 注意事项

- 避免大面积纯白，使用浅灰/浅蓝
- 图标保持简洁，避免细节在深色下丢失
- 文字与背景对比度 ≥ 4.5:1
