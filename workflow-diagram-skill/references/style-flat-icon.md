# Flat Icon 风格规范

默认风格。适合文档、PPT、技术博客。

## 颜色

```
背景:       #ffffff
容器背景:   #f8fafc
容器边框:   #e5e7eb
节点填充:   #ffffff
节点边框:   #d1d5db
主文字:     #111827
副文字:     #6b7280
主流程箭头: #2563eb
迭代箭头:   #9333ea
```

## 节点样式

- 圆角矩形，rx=8, ry=8
- 左侧 56px 处放置圆形图标背景
- 标题 15px 加粗，副标题 12px 灰色

## 图标背景色

| 语义 | 填充 | 描边 |
|------|------|------|
| 用户/输入 | #eff6ff | #2563eb |
| AI/思考 | #faf5ff | #9333ea |
| 代码/生成 | #f0fdf4 | #16a34a |
| 运行/预览 | #fff7ed | #ea580c |
| 检查/验收 | #fef2f2 | #dc2626 |
| 循环/反馈 | #f0fdfa | #0d9488 |
| 部署/上线 | #f0fdfa | #0d9488 |

## 箭头

- 主流程：实线，stroke-width=2，蓝色
- 迭代/反馈：虚线，stroke-dasharray="6,4"，紫色
- marker-end 使用对应颜色箭头

## 容器

- 虚线圆角矩形，stroke-dasharray="6,4"
- 标题 14px 加粗，置于容器内左上角
- 同一阶段内的节点水平排列
