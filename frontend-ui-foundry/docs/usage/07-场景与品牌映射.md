## 选品牌决策

### 按"想要什么感觉"
- 极简克制 → Linear / Vercel / Notion
- 戏剧性 → Apple / Lovable
- 友好亲和 → Stripe / Airtable
- 工程感 → Supabase / PostHog / Sentry
- 信任稳重 → Revolut / Wise / IBM
- 创意活力 → Figma / Miro / Replicate

### 按"避免什么反射"
- 不想"医疗=白+青绿" → 选 Figma 多彩 / Apple 极简
- 不想"金融=深蓝+金" → 选 Revolut 黑紫 / Stripe 暖橙
- 不想"AI 工具=SaaS 米白" → 选 Lovable 紫黑 / Claude 焦橙

## 用法

```bash
# 场景默认品牌
forge admin-dashboard  # 用 2B 灰蓝
forge landing-marketing --brand stripe  # 覆盖为 Stripe 暖色

# 套用品牌到现有项目
brand stripe /path/to/project

# 列出所有品牌
brand list
```
