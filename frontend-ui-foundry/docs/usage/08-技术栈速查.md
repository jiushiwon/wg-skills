# 08 · 6 技术栈关键约定速查

| 栈 | 关键约定 | Token 接入点 |
|----|---------|-------------|
| **html-tailwind** | utility-first + 自定义 @apply | `assets/css/tokens.css` + `tailwind.config.js` |
| **react-nextjs** | App Router + RSC 默认 | `app/globals.css` 引入 + `tailwind.config.ts` extend |
| **vue-nuxt** | Composition + `<script setup>` | `assets/css/tokens.css` + `nuxt.config.ts` css |
| **uniapp** | rpx + 平台条件编译 | `App.vue` 全局 @import + `uni.scss` 变量 |
| **react-native** | StyleSheet.create + Platform.select | `src/theme/colors.ts` + `useTheme()` |
| **svelte** | 极简响应式 + store | `app.css` 全局 + Tailwind config |

## 初始化

```bash
# HTML + Tailwind
npm i -D tailwindcss postcss autoprefixer && npx tailwindcss init -p

# React + Next.js
npx create-next-app@latest my-app --typescript --tailwind --app

# Vue + Nuxt
npx nuxi@latest init my-app && npx nuxi module add @nuxtjs/tailwindcss

# uniapp
npx degit dcloudio/uni-preset-vue#vite-ts my-project

# React Native
npx @react-native-community/cli@latest init MyApp

# Svelte
npx sv create my-app  # 选 SvelteKit + TypeScript + Tailwind
```

## Token 接入

### HTML + Tailwind
```js
// tailwind.config.js content
content: ['./**/*.html', './components/**/*.html']
```
```css
/* assets/css/input.css */
@import './tokens.css';
@tailwind base; @tailwind components; @tailwind utilities;
```

### React + Next.js
```ts
// app/layout.tsx
import './globals.css';  // 引入 @tailwind + tokens
```
```ts
// tailwind.config.ts
theme: { extend: { colors: tokens.colors } }
```

### Vue + Nuxt
```ts
// nuxt.config.ts
css: ['~/assets/css/tokens.css', '~/assets/css/main.css'],
modules: ['@nuxtjs/tailwindcss', '@nuxtjs/color-mode'],
```

### uniapp
```scss
/* uni.scss */
$primary: #c45c48;
$bg-page: #fafaf5;
```
```vue
<!-- App.vue -->
<style lang="scss">
@import '@/static/css/tokens.css';
page { background: var(--color-surface); }
</style>
```

### React Native
```ts
// src/theme/colors.ts
export const colors = { light: { primary: '#c45c48' }, dark: {...} };
```
```ts
// src/theme/useTheme.ts
export function useTheme() {
  const scheme = useColorScheme();
  return { colors: scheme === 'dark' ? colors.dark : colors.light };
}
```

### Svelte
```svelte
<!-- src/routes/+layout.svelte -->
<script>import '../app.css';</script>
```

## 性能优化关键词

| 栈 | 关键 |
|----|------|
| html | 字体预加载 + WebP + lazy load |
| next | `<Image>` + `next/font` + RSC + `<Link prefetch>` |
| nuxt | `<NuxtImg>` + `@nuxtjs/google-fonts` + `useFetch` |
| uniapp | 分包 + rpx + 条件编译 |
| rn | `expo-image` + Reanimated 3 + Hermes + FlatList 虚拟化 |
| svelte | SvelteKit SSR + `<enhanced:img>` + `data-sveltekit-preload-data` |

## 验证清单（共同）

- [ ] 触摸目标 ≥44pt
- [ ] 字号 ≥16px（移动）/ ≥12pt（原生）
- [ ] 减弱动效支持
- [ ] 暗色模式支持
- [ ] viewport 配置正确
- [ ] 安全区处理（iOS / 移动 Web）

## 跨栈组件对照

| 组件 | Web (HTML/Vue/React/Svelte) | React Native | uniapp |
|------|---------------------------|--------------|--------|
| 按钮 | `<button>` | `<Pressable>` | `<button>` / `<view @click>` |
| 文本 | `<p>` / `<span>` | `<Text>` | `<text>` |
| 容器 | `<div>` | `<View>` | `<view>` |
| 滚动 | `overflow: auto` | `<ScrollView>` | `<scroll-view>` |
| 列表 | `ul/li` | `<FlatList>` | `<view v-for>` |
| 输入 | `<input>` | `<TextInput>` | `<input>` |
| 图片 | `<img>` | `<Image>` | `<image>` |
