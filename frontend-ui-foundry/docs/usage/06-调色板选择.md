## 决策树

### 按"承诺度"选
- **Restrained**（克制，主色 ≤10%）：开发者工具、文档、企业 SaaS
- **Committed**（主色占 30-60%）：品牌驱动、支付、文创
- **Full**（4 角色齐备）：数据可视化、监控
- **Drenched**（表面即是色）：3D、消费电子、品牌活动

### 按"主题"选
- **浅色为主**：文档、企业官网、电商
- **深色为主**：监控、3D、游戏
- **双主题**：开发者工具、移动 App（系统跟随）

### 按"品类反思维"避坑
- ❌ 医疗=白+青绿 / 金融=深蓝+金 / 可观测=深蓝 — 这是训练数据反射
- ✅ 重写"谁、哪、什么光、什么心情"物理场景句
- ✅ 选承诺度而非"品类默认"

## 完整角色映射（每套 13 角色）

每套调色板都包含：
- primary / primary-hover / primary-foreground
- secondary / secondary-hover
- accent / accent-foreground
- surface / surface-tinted / surface-elevated
- on-surface / on-surface-muted
- border / border-strong
- error / success / warning / info（含 -foreground）

完整 OKLCH 值见 `references/tokens/color-palettes.md`。

## 暗色模式派生

不直接反转。规则：
- surface 提亮 0.18（用专用 token，不复用浅色）
- on-surface 提亮 0.95
- primary lightness +0.10、chroma 不变
- 单独验对比度

## 在 Tailwind 中使用

```ts
// tailwind.config.ts
colors: {
  primary: { DEFAULT: 'oklch(0.55 0.16 28)', hover: 'oklch(0.48 0.18 28)' },
  surface: { DEFAULT: 'oklch(0.97 0.01 75)', tinted: 'oklch(0.94 0.03 75)' },
  // ...
}
```

完整模板见 `templates/tokens.tailwind.tpl`。
