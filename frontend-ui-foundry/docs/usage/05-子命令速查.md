| `tokens` | Token | 提取/注入/导出/验证 | [04-tokens-管理.md](04-tokens-管理.md) |

## 路由规则

1. **无参数** → 渲染子命令菜单
2. **首词匹配** → 加载 `references/commands/<cmd>.md` 执行
3. **首词不匹配** → 视为通用设计请求：识别场景 → 选风格 → 加载 Token → 实施

## 公共选项

| 选项 | 适用 | 说明 |
|------|------|------|
| `--brand <name>` | forge/optimize/brand | 品牌名（见 [07](07-场景与品牌映射.md)） |
| `--stack <name>` | forge/stack | 技术栈（见 [08](08-技术栈速查.md)） |
| `--dark` | forge | 暗色模式优先 |
| `--prose-only` | forge | 只输出文案 + 骨架 |
| `--strategy <s>` | optimize/unify | conservative / gradual / reshape |
| `--report-only` | optimize/audit | 只输出报告 |
| `--dry-run` | optimize/unify | 试运行不写文件 |
| `--depth <d>` | audit | quick / standard / deep |
| `--focus <f>` | audit | design / code / all |
